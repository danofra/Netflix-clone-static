# Netflix Clone

This is a static clone project of Netflix, developed as part of my study journey. The main objective was to practice front-end technologies like HTML, CSS, Bootstrap, and Sass, as well as to enhance my skills in user interface design.

## Description

The Netflix Clone project is a static replica of the famous Netflix streaming site. Despite being static, it aims to capture the look and feel of the original site, providing smooth navigation and engaging content presentation.
